Introduction

Untyped λ-calculus is a model of computation equivalent to Turing Machines. Its a very simple language with three constructs: variables, applications and λ-terms. The aim of this programming assignment is to implement the α-equivalence between λ-terms [Hindley and Seldin, 2008, Def 1.17].

Assignment (80%)

  • Download the GitHub repository 2016-2-lab3.
    • The module LamTerm.hs defines a data λ-terms.
  • (70%) Using the module LamTerm.hs 
            alphaEq :: Term -> Term -> Bool
    that takes two λ-terms and tells if they are α-equivalent or not, as described by definition 1.17.
  • (10%)
    • Annotate your source code using Haddock. Documentation should explain your solution for the assignment.
    • Improve your implementation with HLint.
